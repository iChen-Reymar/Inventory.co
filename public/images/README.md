# Product Images

Place your product images in this folder with the following names:

- `bass-guitar.jpg` - Bass guitar image
- `acoustic-guitar.jpg` - Acoustic guitar image
- `keyboard.jpg` - Piano keyboard image
- `electric-guitar.jpg` - Electric guitar image
- `drum.jpg` - Drum set image

The images will be automatically loaded in the Products page. If an image is missing, a placeholder will be shown instead.
